===============================
📄 COMANDOS BÁSICOS DO GIT
===============================

🔄 CLONAR UM REPOSITÓRIO
-----------------------------------
 - git clone http://seu_repositorio...
Clona um repositório remoto para sua máquina local.


🔄 ATULIZAR MEU REPOSITÓRIO
-----------------------------------
 - git clone pull
Clona um repositório remoto para sua máquina local.

📊 VERIFICAR STATUS DO REPOSITÓRIO
-----------------------------------
 - git status
Exibe a situação atual do repositório, como arquivos modificados, adicionados ou prontos para commit.

➕ ADICIONAR ARQUIVOS AO STAGE
-----------------------------------
 - git add *
Adiciona todos os arquivos modificados ao stage (preparando para commit).
💡 Dica: Prefira `git add .` em vez de `git add *`, pois ele inclui arquivos ocultos também.

📝 CRIAR UM COMMIT
-----------------------------------
 - git commit -m "Descrição das alterações"
Salva as alterações adicionadas com uma mensagem descritiva.

🚀 ENVIAR PARA O REPOSITÓRIO REMOTO
-----------------------------------
 - git push
Envia os commits locais para o repositório remoto (ex: GitHub).
